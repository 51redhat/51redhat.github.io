更换theme主题

2018-03-10

==主题安装==

如果觉得默认的landscape并不是自己最喜欢的那一款，我们可以去https://hexo.io/themes/中挑选喜欢的主题。
进入到个人主题的theme库中clone or download
选择复制copy to clipboard这个主题地址，下面就以 Noise 为例。

==Step== 1:

在theme主题的下载地址hexo-theme-noise中的noise字样就是对应的主题名称。

==Step== 2:

根据 Noise 主题 上的文档，我们需要先安装主题和渲染器：

  • 1. git clone https://github.com/lotabout/hexo-theme-noise themes/noise
  • 2. npm install hexo-renderer-less –save
  • 3. npm install hexo-renderer-jade –save

==Step== 3: 根据 Noise 主页 上的文档，将 your_name.github.io/_config.yml 中的 theme 的值由原来默认的 landscape 改为 noise 。

至此，主题已经更换完成，依次git上运行命令：

  • hexo generate
  • hexo deploy
  • hexo server

我们可以在 http://localhost:4000/ 上查看我们网站新的样式。根据 Noise 主页 上的文档，我们还可以进一步的配置该主题，不过这内容就扯开去了。过段时间，我会专门写一篇文章关于 Noise 主题的配置、对 LaTeX 的支持和访问速度的优化，敬请期待。更换其他主题的操作与此基本相同，不再赘述。
